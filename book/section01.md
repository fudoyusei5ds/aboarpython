# 文件树与helloworld

## 1.linux操作系统的文件树
### 简介
linux的目录(directory)架构是以根目录为主, 向下呈现分支状的目录结构的一种文件架构. 根目录用"/"表示.  
### 路径与工作目录
路径: 从根目录到指定目录的所有节点记录.  
工作目录: 当前用户所在的节点.  
### 目录操作 
cd "directory": cd(change directory)的缩写, 可以将当前工作目录切换为指定路径.  
mkdir "new": mkdir(make directory)的缩写, 创建一个新的目录. 
pwd: (print work directory)的缩写, 打印工作目录.
